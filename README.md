# yomitai
CLI Application for quick reference of Japanese-Yomikata.

## Usage
<pre>
<code>$ yomitai 劈く 
つんざく
</code></pre>

<pre>
<code>$ yomitai --kana 齟齬
ソゴ
$ yomitai --k 齟齬
ソゴ
</code></pre>

<pre>
<code>$ yomitai --romaji 炙りカルビ
aburi karubi
$ yomitai --r 炙りカルビ
aburi karubi
</code></pre>
